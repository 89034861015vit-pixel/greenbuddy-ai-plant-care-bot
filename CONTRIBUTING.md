# Contributing to GreenBuddy

Thank you for your interest in contributing to GreenBuddy.

GreenBuddy is an open-source learning-focused Telegram AI bot for plant care. The project is designed to demonstrate Vision AI, Telegram bot UX, SQLite logging, chart generation, and maintainer workflows.

## Ways to Contribute

You can help by:

- improving documentation
- adding tests
- improving error handling
- improving security practices
- adding demo screenshots
- suggesting better plant care workflows
- improving code structure

## Development Principles

Please keep contributions:

- clear
- beginner-friendly
- secure
- documented
- focused on practical user value

## Before Opening a Pull Request

Please make sure that:

- no real API keys or tokens are committed
- `.env.example` is used for example variables
- code is readable and documented
- changes match the project roadmap
- new features are described clearly

## Current Priorities

- add unit tests
- improve Vision API error handling
- add GitHub Actions
- add demo screenshots
- improve security checks
- improve documentation

## Maintainer Note

This project is maintained by a beginner developer who is learning in public and building practical open-source AI tools. Contributions, suggestions, and educational feedback are welcome.
