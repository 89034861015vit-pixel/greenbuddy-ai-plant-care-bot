# Security Policy

GreenBuddy is an open-source learning project focused on Telegram AI agents, Vision AI, SQLite logging, and safer development workflows.

## Supported Version

The current supported version is:

- v0.1.0-mvp

## Reporting Security Issues

Please do not publish security issues publicly.

If you find a vulnerability, please open a private discussion with the maintainer or contact the project owner through GitHub.

## Security Scope

This project may involve:

- Telegram bot tokens
- API keys
- User-generated photo inputs
- SQLite data storage
- Third-party Python dependencies

## Current Security Goals

- Keep secrets out of the repository
- Use `.env.example` instead of real environment files
- Improve input validation
- Improve dependency safety
- Add automated checks in future releases
- Use Codex Security to detect unsafe patterns and improve code quality

## Maintainer Note

This project is maintained as a practical learning-focused open-source AI agent. Security improvements are part of the roadmap.
