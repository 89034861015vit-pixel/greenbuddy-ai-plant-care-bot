# Roadmap

## Current MVP

- Telegram bot interface
- Plant photo analysis
- Structured care recommendations
- SQLite logging
- Basic statistics command
- Health chart generation

## Next Steps

- Add unit tests
- Add better Vision API error handling
- Add reminder scheduling
- Improve plant profiles
- Add CSV/PDF export
- Add GitHub Actions workflow
- Improve documentation

## Long-Term Vision

GreenBuddy can become an open-source reference project for practical AI agents in everyday life: photo understanding, user memory, habit loops, and lightweight automation.
